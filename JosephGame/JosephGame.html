约瑟夫生者死者游戏

数据结构课程设计作业

  1. 简介

    2. 本程序基于命令行界面，向用户提供了约瑟夫生者死者游戏的操作接口。

    本系统使用C++语言写成，可以在任何支持C++14标准的编译器上编译通过。

  2. 功能

    3. 这个程序可以实现约瑟夫生者死者游戏的功能，由用户输入总人数、间隔人数和期望的存活人数，系统输出死亡顺序。

    每个乘客都用一个数字表示，从1到n排列。

  3. 实现

    4. 程序没有用链表模拟实现，而是通过迭代的方式计算出一个死亡的顺序表。提供的接口就通过这个表进行。

    用图来说明逻辑如下：

    如图，总人数为9，如果我们设定间隔为7，剩余人数为2，那么死亡的玩家应该为：

    顺序为7 5 4 6 9 3 8。

  4. API

    5. API结构非常简单紧凑。

    class JosephCircle {

        // 构造函数，构造完毕后结果就已经生成了
        JosephCircle(unsigned n_people, unsigned len_section, unsigned n_alive, unsigned start = 1)

        // 数组包含了幸存者的编号
        const std::vector<unsigned>& livings;

        // 数组包含了死亡者的编号，按顺序
        const std::vector<unsigned>& died;

    };
